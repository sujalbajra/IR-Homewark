Wildcards allow you to search for variations of a word using the * character.

  • Trailing Wildcard (kth*): Matches terms starting with 'kth' (e.g., kathamandu, katha).
  • Leading Wildcard (*mandu): Matches terms ending with 'mandu' (e.g., kathmandu).
  • Middle Wildcard (k*mandu): Matches terms with 'k' start and 'mandu' end.
